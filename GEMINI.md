# Project Name: we-wizards

## Project Goal
プログラミング学習をゲームのように楽しくするために、GitHubの活動をRPGのステータスとして可視化するnpmライブラリを開発する。
利用者は本ライブラリをGitHub Actionsと組み合わせて、自身のプロフィールREADME.mdを定期的に自動更新することを想定している。
また、外部ライブラリへの依存を極力減らし、最小限の構成でシンプルなつくりを目指す。

## Core Features
1.  **GitHubデータ取得:**
    *   ユーザーのコミット履歴やPublicリポジトリにおける言語のバイト情報を取得する。
    *   GitHub APIの利用を想定。
2.  **ステータス算出ロジック:**
    *   取得したGitHubデータ（コミット数、コード変更量、言語利用状況など）を元に、以下の基準で「魔法攻撃力」「魔法防御力」を算出する。
        *   **魔法攻撃力:** `tests/`, `test/`, `docs/`, `doc/` 配下にないファイルの総バイト数。
        *   **魔法防御力:** `tests/`, `test/`, `docs/`, `doc/` 配下にあるファイルの総バイト数。
3.  **レベル・称号システム:**
    *   総バイト数（経験値）に応じてレベルが上昇する。レベルアップに必要な経験値は、高レベルになるほど増加する。レベルの上限は設けない。
    *   総バイト数に応じて10段階の称号が付与される。一定のバイト数を超えるごとに称号が更新される。
4.  **npmライブラリとしての提供:**
    *   簡単にプロジェクトに導入できるnpmパッケージとして公開する。

## Technical Stack

-   Node.js
